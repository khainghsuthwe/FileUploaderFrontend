# FileUploader Frontend

This is the Nuxt 2 + Vuetify frontend for the FileUploader take-home assignment.

Requirements

- Node 18+

## Demo

- Demo application hosted [link](https://file-uploader-frontend-kinobi.vercel.app/)
- Repository [link](https://github.com/khainghsuthwe/FileUploaderFrontend)

## Run locally (development)

1. Clone the Repository

```bash
git clone https://github.com/khainghsuthwe/FileUploaderFrontend.git
```

1. Install dependencies

```bash
cd FileUploaderFrontend
npm install
```

2. Create .env file and add the following:

```bash
# Set API_URL to point to your backend (default fallback: http://localhost:5001)
API_URL="http://localhost:5001/api"
```

3. Start dev server

```bash
 npm run dev
```

Build & Start

```bash
npm run build
npm run start
```

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
